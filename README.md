### Comandos básicos de terminal

Lista o coteúdo de um diretório (pasta)

```
dir
```

Criar um novo diretório

```
mkdir <nome-da-pasta>
```

Remover um diretório

```
rmdir <nome-da-pasta>
```

Acessar uma pasta

```
cd <nome-da-pasta>
```
